CoinvidBot Setup Guide

This guide provides step-by-step instructions to set up the CoinvidBot project on a Linux system (Ubuntu/Debian).

Prerequisites

  • A Linux-based system (e.g., Ubuntu/Debian).
  • Root or sudo privileges.
  • Internet connection for downloading packages and cloning the repository.

Setup Instructions

1. Update Package Lists

Refresh the system's package repository.

sudo apt update

2. Upgrade System Packages

Update all installed packages to their latest versions.

apt update && apt upgrade -y

3. Install Python and Development Libraries

Install Python 3, pip, and required development libraries.

apt install python3 python3-pip libssl-dev libffi-dev -y

4. Install python-socks

Install the python-socks library for SOCKS proxy support.

pip install python-socks

5. Install Git

Install Git for version control.

sudo apt install git

6. Install tmux

Install tmux for managing terminal sessions.

sudo apt install -y tmux

7. Verify Installed Versions

Check the versions of installed tools.

git --version
python3 --version
pip3 --version
tmux -V

8. Install Python Dependencies

Install required Python libraries for the bot.

pip3 install python-telegram-bot dotenv stem requests

9. Install Tor

Install Tor for anonymous network communication.

apt install tor -y

10. Start Tor Service

Launch the Tor service.

systemctl start tor

11. Enable Tor on Boot

Configure Tor to start automatically on system boot.

systemctl enable tor

12. Check Tor Status

Verify that the Tor service is running.

sudo service tor status

13. Test Tor Connectivity

Check your public IP through Tor's SOCKS5 proxy.

curl --socks5 localhost:9050 https://api.ipify.org

14. Restart Tor Service

Restart Tor to apply changes.

sudo service tor restart

15. Clone CoinvidBot Repository

Download the CoinvidBot source code from GitHub.

git clone https://github.com/Daicool/CoinvidBot.git

16. Set Timezone

Configure the system timezone to Asia/Ho_Chi_Minh.

sudo timedatectl set-timezone Asia/Ho_Chi_Minh

17. Create a tmux Session

Start a new tmux session named "CoinvidSession".

tmux new -s CoinvidSession

18. Split tmux Window

Split the tmux window horizontally.

tmux split-window -h

19. Reattach to tmux Session

Reconnect to the "CoinvidSession".

tmux attach-session -t CoinvidSession

Next Steps

  • Navigate to the CoinvidBot directory (cd CoinvidBot).
  • Configure environment variables (e.g., Telegram bot token) in a .env file.
  • Run the bot using the appropriate command (refer to the project's documentation).

Troubleshooting

  • If Tor fails to connect, check the service status with sudo service tor status and ensure port 9050 is open.
  • Verify Python dependencies with pip3 list.
  • For tmux issues, list active sessions with tmux list-sessions.

Happy coding with CoinvidBot!
